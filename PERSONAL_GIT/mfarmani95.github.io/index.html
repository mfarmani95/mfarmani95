About

I study land–atmosphere processes and hydrologic modeling with a focus on Noah-MP / RAPID, evaluating precipitation forcings (AORC, NLDAS2, CONUS404, IMERG) against gauges, and building differentiable PyTorch components for soil moisture and water table dynamics.

Tooling: Python (xarray, rioxarray, geopandas, pandas, PyTorch), Fortran, HPC.
